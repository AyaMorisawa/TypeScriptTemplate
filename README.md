# TypeScript Template
[![][travis-badge]][travis-link]

## How to build
1. `git clone git://github.com/AyaMorisawa/TypeScriptTemplate.git`
2. `cd TypeScriptTemplate`
3. `npm install`
4. `npm run dtsm`
5. `npm run build`
